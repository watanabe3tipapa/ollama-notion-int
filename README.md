# Ollama Notion Int

Web UI to generate text with Ollama and save to Notion.

## Quick Start

```bash
# Create venv
uv venv

# Install dependencies
uv pip install -r requirements.txt

# Run
.venv/bin/python app.py
```

Open http://localhost:7860
